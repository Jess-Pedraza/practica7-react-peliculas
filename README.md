## Objetivo

Desarrollar una aplicación web moderna implementando componentes, consumo de API, navegación con React Router y despliegue en Netlify.

- React
- Vite
- React Router DOM
- CSS
- TMDB API
- GitHub
- Netlify

## Instalación y ejecución local

1. Clonar repositorio:
git clone URL

2. Entrar al proyecto:
cd practica7-react-peliculas

3. Instalar dependencias
npm install

4. Ejecutar proyecto
npm run dev

## Rutas utilizadas

- / → muestra listado de películas
- /movies/:movieId → muestra el detalle de la película

## API utilizada

Se utilizo TMDB API (The Movie Database) para tener informacion de peliculas

## Descripción

La aplicación muestra películas populares obtenidas desde la API TMDB. Cada película incluye imagen, título y calificación. Al seleccionar una película se muestra información detallada como descripción, géneros, fecha de estreno y puntuación.
